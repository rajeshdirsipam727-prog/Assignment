# AI Image & Text Generator

## Project Overview
This is a simple webpage prototype that supports:

- Text prompt enhancement
- AI image generation
- Image analysis workflow

## Features

### 1. Text Workflow
- User enters a text prompt
- System enhances the prompt
- Generates an AI image

### 2. Image Workflow
- User uploads an image
- System analyzes the image and returns a caption

## Technologies Used
- HTML
- CSS
- JavaScript
- Hugging Face API

## Files
- index.html
- style.css
- script.js

## How to Run
1. Open the project in VS Code
2. Add your Hugging Face API key in `script.js`
3. Run with Live Server

## API Used
- Stable Diffusion (Image Generation)
- BLIP Image Captioning (Image Analysis)
